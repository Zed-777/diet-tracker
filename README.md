# Diet Tracker

A single-page diet and meal tracker app in `diet-tracker.html`.

## Deploying on Netlify

This project can be deployed directly from this repository to Netlify. The HTML file is self-contained and requires no build step.

## Notes

- Uses Supabase REST API for persistence
- The app is already deployed on Netlify
- Keep the Supabase credentials and table setup secure if this repo is shared publicly
